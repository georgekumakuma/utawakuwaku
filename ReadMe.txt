# utawakuwaku　～うたわくわく！～

YouTubeのプレイリスト区間再生・評価管理Webアプリ／PWA

---

## ■ アプリの特徴

- 好きなYouTube動画をCSVでまとめて管理し、区間指定再生
- 評価（★）やコメントも曲ごとに保存
- スマホやPCで「ホーム画面に追加」→アプリのように利用可能
- 完全ローカル動作・ダウンロード不要・ログイン不要
- プレイリスト（CSV）はテキストエディタで直接編集OK

---

## ■ 使い方

1. [https://utawakuwaku.netlify.app] へアクセス
2. スマホなら「ホーム画面に追加」してPWAとして使えます
3. プレイリストCSVをインポート／エクスポート可能
　　・初回起動時はサンプルが表示されます（ぜひ聴いてみてください！）
    - スマホの場合、ファイルの保存場所・選択方法にご注意ください（「ファイル」アプリや「ダウンロード」フォルダから選択）

---

## ■ セキュリティ・注意事項

- 本アプリはYouTube公式APIのみを利用
- 動画ダウンロードや不正な機能は一切ありません
- プレイリスト共有時は、個人情報や非公開URL等を含めないようご注意ください
- ソースコードは自由に参照・改変できますが、派生版の配布は自己責任でお願いします
