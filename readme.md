# 🎵 utawakuwaku - Improved Version

YouTube動画区間再生プレイリストアプリの改良版です。

## ✨ 新機能・改善点

### 🎵 音響的改善
- **スムーズなフェード機能** - 曲の切り替え時に自然なフェードイン/アウト
- **正確な区間再生** - より精密な時間監視とシーク調整
- **ボリューム制御** - リアルタイムボリューム調整機能

### 🎨 現代的なUI/UX
- **モダンなデザイン** - CSS変数、グラデーション、シャドウを活用
- **レスポンシブデザイン** - モバイル対応の改善されたレイアウト
- **インタラクティブな要素** - ホバー効果、アニメーション、視覚的フィードバック
- **統一されたダークテーマ** - 目に優しい美しいデザイン

### 💫 ユーザビリティ向上
- **トースト通知** - ユーザーアクションに対する明確なフィードバック
- **改良された時間設定UI** - 現在位置ボタン、動画最後ボタン付きポップアップ
- **視覚的プレイリスト** - 番号表示、改善されたアイコン、状態表示
- **空状態の適切な処理** - プレイリストが空の場合のガイダンス

### 🔧 信頼性の改善（v1.1）
- **区間再生の正確化** - 終了タイマーを`setInterval`ベースに変更し、バックグラウンドタブでも終了時間を確実に守るように修正
- **プレイヤー側の区間終了強制** - `endSeconds`をYouTubeプレイヤーに渡す二重の保険
- **曲切替レースの解消** - 再生世代トークンで素早い曲切替時の誤タイマー発火を防止
- **再生エラーの可視化＆自動スキップ** - 埋め込み不可・削除済み動画をトーストで通知し、次の曲へ自動スキップ（「YouTubeで開く」ボタン付き）
- **シーク挙動の予測可能化** - 区間内シークでは終了時間を守り、終了位置を越えてシークした時のみ自動終了を解除

### 📱 モバイル最適化
- **スマホ縦画面でシーン名を最優先表示** - 動画IDの代わりにサムネイル＋シーン名＋時間を表示

### 🎛 モダン機能
- **複数プレイリスト** - 作成・切替・名前変更・削除に対応、すべて自動保存
- **サンプルプレイリストバンク** - 同梱の複数サンプルCSV（`playlists/samples/`）から新規プレイリストとして読み込み可能
- **共有URL** - プレイリストをURLひとつで共有（サーバー不要、URLを開くだけでインポート）
- **サムネイル表示** - プレイリスト各行に動画サムネイル
- **Media Session API** - OSのメディアキー・ロック画面・イヤホンボタンから再生操作
- **進捗バーのクリックシーク** - 区間内の任意の位置へジャンプ

## 📺 YouTube Premium・広告表示について

このアプリは YouTube公式の IFrame Player API を使い、実際の `youtube.com` の埋め込みプレイヤーをそのまま表示しています。動画をダウンロードしたり別サーバーを経由したりはしないため、**広告が出るかどうかは完全にYouTube側・ブラウザ側の判断**であり、本アプリのコードで制御することはできません。仕組み上は次のようになっています。

- 埋め込みプレイヤー（iframe）は、そのページを開いているブラウザが保持する `youtube.com` の**Cookie（ログインセッション）**を使って動画をリクエストします。そのCookieが有効なPremiumアカウントのものであれば、埋め込みでも広告なしで再生されます（これはYouTubeの公式な埋め込み仕様であり、通常のyoutube.comページと同じ挙動です）。
- 逆に、そのCookieが送られない・存在しない場合は「ログインしていない状態」として扱われ、広告が表示されます。**これはキャッシュの問題ではなく、Cookie／ログインセッションの問題です。**キャッシュを消しても解決しません。

### ブラウザ版（`index.html` を直接、または `node server.js` 経由で開く場合）
同じブラウザ・同じプロファイルでYouTube Premiumにログイン済みなら、基本的にはそのまま広告なしで再生されるはずです。ただし以下のいずれかに該当すると、ログイン状態が埋め込みプレイヤーに伝わらず広告が出ることがあります。

- ブラウザの「サードパーティCookieをブロック」設定が有効（Chrome/Edge/Firefoxとも近年デフォルトでブロックへ移行中）
- シークレットモード／プライベートブラウジングで開いている
- YouTubeにログインしていない別のプロファイルで開いている

**対処法**: ブラウザの Cookie 設定で `[*.]youtube.com` を「常に許可」する例外を追加してください（例: Chrome の場合 `chrome://settings/content/cookies` → 「常に許可するサイト」に `[*.]youtube.com` を追加）。このアプリを `file://` で直接開くよりも、同梱の `node server.js` を起動して `http://localhost:3000` から開く方が、Cookie関連の挙動が安定します。

### デスクトップアプリ版（Electronの `.exe` 版）
**こちらは仕組み上、Premiumでも常に広告なしにはなりません。** Electronアプリのウィンドウは、普段お使いのChrome等とは完全に別の独立したブラウザプロファイル（Cookieストレージ）を持っており、最初から一度もYouTubeにログインしていない状態です。さらに、アプリ内にURLバーやログイン画面がないため、そもそもログインする手段がありません。

加えて、仮にログイン画面を組み込んだとしても、Googleは Electron のような「組み込みブラウザ（embedded user agent）」からのGoogleアカウントへのサインインをセキュリティ上の理由で明示的にブロックしており（"This browser or app may not be secure" エラー）、正攻法での回避はできません。

**そのため、Premium会員としての広告なし再生を確認・利用したい場合は、Electronの `.exe` 版ではなく、普段YouTube Premiumにログインしているブラウザで直接このアプリを開く（ブラウザ版）ことを強くおすすめします。**

## 🚀 使い方

1. `index.html`をブラウザで開く
2. YouTube URLまたは動画IDを入力
3. 開始・終了時間を設定してプレイリストに追加
4. 改良されたUIで快適な音楽体験をお楽しみください！
